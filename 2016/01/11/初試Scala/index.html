感想

很複雜的語言……很多feature(果然是教授的產物)

除了method 外可以自動估算data type
val r = 1
Object 像一Java 的static method, 不用new object 便可以使用

AnyVal

AnyVal is primitive datatype in Java
底下有

  • Int
  • Double
  • Flot
  • Long
  • Short
  • Byte
  • Char
  • Boolean
  • Unit (Single Value, ~Void in Java)

AnyRef

  • List

Method

定義

method 定義有點像haskell,
名在前, data type 在後
= 之後是回傳數值
def hi(a: String, b: String) = a + b

多種括號
def hi(a: String)(b: String) = a + b
可以這樣呼叫
hi("hi")"Jason"
or
hi("hi") {"jason"}

Accept an function

functional map on array

強大的pattern matching

相比java 麻複的regex……

  • r 為regex
  • l 為testing string
  • _ 指不在乎
