JSON 可以由Context-free Grammar 所做成。當中的rule都是non-ambiguous，內裡的leftmost derivation, 都是deterministic (只有一個方法)。而且可以先做rule1 再做rule2。因此他是LL(1) parser, 可以用predictive parsing，用recursive decent method.
